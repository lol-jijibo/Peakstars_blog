# MEMORY.md - 项目长期记忆

## 项目信息
- **项目名称**：面经宝典（Interview-demo）
- **工作目录**：`d:/Frontend-stars/Interview-demo`
- **技术栈**：Vue 3 + Vite 5 + Vue Router 4
- **运行命令**：`npm run dev`（端口 5173）

## 用户偏好与规范
1. **CSS 与 HTML 完全分离**，禁止内联样式，样式统一放独立 CSS 文件
2. **开发模式**：单文件原型优先，后按企业级规范拆分重构
3. **响应式**：移动端 + PC 端兼容，移动端优先
4. **代码组织**：Vue2+3 目录规范（views/、components/、assets/styles/、data/）
