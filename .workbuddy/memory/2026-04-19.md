## 面经宝典项目搭建完成

### 项目路径
`d:/Frontend-stars/Interview-demo`

### 技术栈
- Vue 3 + Vite 5 + Vue Router 4（按 Vue2+3 目录规范组织）
- 纯 CSS 样式，CSS/HTML 完全分离，无内联样式

### 目录结构
```
src/
  main.js          # 入口 + 路由配置
  App.vue          # 根组件
  data/
    interviews.js  # 面经数据（8家企业，前端+Java后端）
  views/
    InterviewList.vue   # 列表页（含 TabBar、搜索、分类筛选）
    InterviewDetail.vue # 详情页（点赞、收藏、返回）
    Collect.vue         # 收藏页
    Like.vue            # 喜欢页
    Mine.vue            # 我的（个人中心）
  components/
    InterviewCard.vue   # 面经卡片组件
  assets/styles/
    global.css          # 全局样式 + 响应式（移动端/PC）
```

### 面经数据
8家企业：百度前端、字节跳动前端、阿里Java、腾讯前端、美团Java、网易前端、京东Java、滴滴前端

### 特性
- 响应式：移动端单列 + PC端多列网格，TabBar PC转侧边栏
- 搜索 + 分类（全部/前端/Java后端）筛选
- 详情页 Markdown 内容渲染（加粗、列表、代码块）
- 点赞/收藏交互（本地状态）

## 面经数据库化改造

### 新增文件
- `database/schema.sql` — MySQL 表结构（6张表：company/category/tag/interview/interview_tag/user_action）
- `database/seed.sql` — 初始数据导入（8家企业面经 + 标签关联）
- `database/README.md` — 数据库接入操作指南
- `server/` — Node.js Express 后端服务（mysql2 连接池）
  - `src/app.js` — 入口 + 路由注册
  - `src/config/db.config.js` — 数据库连接配置（需用户修改密码）
  - `src/db/pool.js` — 连接池
  - `src/controllers/interview.controller.js` — 面经CRUD接口
  - `src/routes/interview.routes.js` — 路由定义
- `src/api/interview.js` — 前端 API 请求封装模块
- `.env.development` — Vite 环境变量（VITE_API_BASE_URL）

### 改造文件
- `src/views/InterviewList.vue` — 改为调用 `/api/interviews` 接口，含防抖搜索、加载/错误状态
- `src/views/InterviewDetail.vue` — 改为调用 `/api/interviews/:id`，点赞/收藏对接后端

### 架构
前端 Vue3 (5173) → Express API (3000) → MySQL (3306/interview_db)

### 注意
- 后端密码默认 `123456`，需修改 `server/src/config/db.config.js`
- 静态数据 `src/data/interviews.js` 保留，可降级使用
